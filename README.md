# TBCS Website

Sitio web base de TechBridge Consulting Systems (`tbcs.com.mx`).

## Estructura

- `index.php`: pagina principal
- `css/`: estilos globales y de paginas
- `js/`: scripts compartidos
- `img/`: recursos visuales del sitio
- `api/`: endpoints de contacto
- `database/`: respaldo SQL base
- `proyectos/nexora-health/`: micrositio integrado de Nexora Health

## Flujo recomendado

1. Trabajar cada cambio en commits pequenos y claros.
2. Mantener `main` como rama base estable.
3. Versionar cambios visuales, contenido y recursos relacionados en el mismo commit cuando formen una sola entrega.

## Notas

- El proyecto usa PHP, CSS y JavaScript sin framework.
- Para entorno local se ha estado usando XAMPP con la ruta `c:\xampp\htdocs\tbcs`.
